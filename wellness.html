功能医学检测类型

基础功能检测

代谢、肝肾功能、炎症基础指标与营养状态评估。

  • CBC / CRP
  • 肝功能 / 肾功能
  • 维生素 / 微量元素

激素与内分泌

甲状腺、性激素、皮质醇节律等，聚焦“平衡与节律”。

  • TSH / T3 / T4
  • 雌激素 / 睾酮 / DHEA
  • 皮质醇节律

血管风险评估

从脂质谱到炎症风险，建立血管健康管理路径。

  • 血脂深度分析
  • 同型半胱氨酸
  • 氧化压力指标

肠道与免疫

菌群、消化吸收与免疫反应，定位不适与慢性问题来源。

  • 肠道菌群分析
  • 食物不耐受
  • 过敏原筛查

压力与睡眠评估

自主神经与恢复能力评估，解决“累但睡不稳”。

  • HRV / 自主神经
  • 睡眠结构分析
  • 压力节律建议

体重与体成分管理

体成分、基础代谢与习惯调整，做长期可持续管理。

  • 体成分 / 基础代谢
  • 饮食结构与运动处方
  • 阶段复测与调整

Signature 3种方案

血管焕新计划

适合：血脂异常、疲惫、循环差、长期久坐与压力人群。

  • 检查：脂质谱 / 炎症与氧化压力 / 风险评估
  • 周期：3–4周为一阶段
  • 输出：饮食结构 + 运动要点 + 复测建议

激素平衡计划

适合：睡眠差、情绪波动、体重难控、精力下降、亚健康人群。

  • 检查：甲状腺 / 性激素 / 皮质醇节律
  • 周期：1个月为一阶段
  • 输出：节律管理 + 习惯清单 + 复测节点

肠道重建计划

适合：腹胀、消化差、皮肤反复、免疫力下降、过敏体质。

  • 检查：菌群 / 食物不耐受 / 免疫与过敏筛查
  • 周期：4–8周为一阶段
  • 输出：饮食结构 + 修复策略 + 随访复盘
  • 用这3种方案定制行程
